# Assets

Generated presentation assets for `claude-research-bot`.

## Files

- `hero.png` - README banner.
- `brand-card.png` - square brand card / social preview.

## Direction

The assets use the local Superuser palette:

| Role | Color |
| --- | --- |
| Background | `#262a33` |
| Elevated surface | `#1f232c` |
| Foreground | `#e5f7ef` |
| Muted text | `#b8c5d0` |
| Accent | `#43ffaf` |
| Border | `#526777` |

They were generated as bitmap assets and should stay committed under stable filenames so the README renders consistently.
